Introduction

Où insérer du Javascript dans une page HTML

  • Directement dans le HTML entre les balises <script> et </script>
  • Mieux : Dans un fichier séparé .js et appelé via l'attribut src de la balise <script>
  • Où : Juste avant la balise de fermeture </body> après que les éléments de la page aient été tous chargés

Page HTML avec script

        <!DOCTYPE html>
        <html lang="fr">
          <head><meta charset="utf-8" /></head>
          <body>
            <!-- Tout votre code HTML -->
            <script>
              document.write( "<h2>Un titre niveau 2</h2>" );
            </script>
          </body>
        </html>

Page HTML avec script

        <!DOCTYPE html>
        <html lang="fr">
          <head><meta charset="utf-8" /></head>
          <body>
            <!-- Tout votre code HTML -->
            <script src="scripts/mon_code.js">
            </script>
          </body>
        </html>

Javascript : un langage (pseudo)-objet

  • Syntaxe ≈ langage Java
  • Un objet : propriétés + méthodes (fonctions) + objets
  • Objet de plus haut niveau hiérarchique dans un navigateur : window
  • window contient l'objet document
  • document fournit un accès à la structure du document HTML

  • etc.

Types, variables et portée

  • Types de base : nombres, chaînes de caractères, booléens
  • Faiblement typé :
    • Type déterminé au moment de l'affectation
    • Une variable peut à tout moment changer de type

Déclaration de variables

  • Déclaration avec (locale) ou sans (globale) mot-clé var
  • Pas de visibilité locale à un bloc
         <script>
           var compteurMax = 12; // valeur entière
           compteurMax = 5.35; // valeur réelle
         </script>

DOM (Document Object Model) du W3C

  • Décrit l'arborescence des imbrications de balises d'un document

DOM : Arborescence

  • Noeuds : balises ("éléments"), attributs, commentaires, etc.
  • Noeuds terminaux (feuilles) : texte
  • Noeuds accessibles via attributs id ou name

DOM : Types de noeuds

Types de noeuds DOM (nodeType)
Type de noeud Constante (numéro) Description
Document Node.DOCUMENT_NODE (0) document (racine du DOM)
Element Node.ELEMENT_NODE (1) balise HTML
Attr Node.ATTRIBUTE_NODE (2) attribut d'un élément
Text Node.TEXT_NODE (3) texte à l'intérieur d'une balise
Comment Node.COMMENT_NODE (8) commentaire HTML
DocumentFragment Node.DOCUMENT_FRAGMENT_NODE (11) sous-arbre de DOM hors document

Javascript et DOM : parents et enfants

    Soit le noeud n
  • son noeud parent var p = n.parentNode
  • ses enfants : var enfants = n.childNodes
  • son premier enfant : n.firstChild ou n.firstElementChild
  • son dernier enfant : n.lastChild ou n.lastElementChild
