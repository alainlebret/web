Introduction

HTTP (HyperText Transfert Protocol)

HTTP (HyperText Transfert Protocol)

  • Protocole de communication client/serveur
  • Utilise TCP (Transfert Control Protocol)
  • Port 80
  • RFC 2616 (1999) et 7230-7240 (2014)
  • Visualiser les requêtes HTTP brutes
    • Greffon LiveHTTPHeaders (Firefox, Chrome )
    • Analyseur de paquets : Wireshark, Kismet, tcpdump

HTTP : Demande de page (requête GET)

GET http://www.lycee-dorian.org HTTP/1.1

HTTP : Envoi de données (requête GET)


Sans modification des données distantes (recherche)

GET http://www.commentcamarche.net/search/index.html?q=java

HTTP : Envoi de données (requête POST)


Avec modification des données distantes (formulaire)

POST /www.commentcamarche.net/register/_xhr__signup/

HTTPS : Un HTTP sécurisé

  • Protocole HTTP
  • Couche de chiffrement TLS* (Transport Secure Layer) ou SSL (Secure Socket Layer) + Certificat
  • Port 443
  • RFC 2818 (2000)

(*) Version actuelle : 1.2 (1.0 vulnérable)
