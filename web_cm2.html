Introduction

HTML (HyperText Markup Language)

Gérer et organiser le contenu d'une page Web : texte, liens, images, etc.

  • HTML 1 (1991) Première version créée par Tim Berners-Lee
  • HTML 2 (1994) Règles et fonctionnement W3C *
  • HTML 3 (1996) Ajout des tableaux, applets, scripts, etc.
  • HTML 4 (1998) frames, améliorations tableaux et formulaires, feuilles de style
  • HTML 5 (?) Inclusion de vidéos, amélioration de l'agencement, etc.

(*) World Wide Web Consortium

CSS (Cascading Style Sheet *)

Gérer l'apparence d'une page Web : agencement, positionnement, décoration, couleurs, taille du texte, etc.

  • CSS 1 (1996) Couleurs, marges, polices de caractères, etc.
  • CSS 2 (1999) Amélioration du positionnement
  • CSS 3 (?) Bordures arrondies, dégradés, ombres, etc.

(*) Feuille de style en français

Bases HTML

Balises HTML

  • Les pages HTML sont formées de balises invisibles dans le navigateur
  • Les balises sont entourées de chevrons < et >
  • Les balises indiquent la nature du texte qu'elles encadrent : titres, paragraphes, images, etc.
  • Deux types : les balises en paires et les balises orphelines

Balises HTML en paires

  • Elles encadrent du texte et indiquent sa nature
		  <h1>Ceci est un titre très important</h1>

Balises HTML orphelines

  • Elles permettent d’insérer un élément à un endroit précis
		  <image />

Attributs de balises

  • Les attributs fournissent des options aux balises en paires ou orphelines
		  <image src="photo.jpg" alt="" />
		
		  <q cite="http://exemple.org/boileau/2345.html">Ce qui
                se conçoit bien s'énonce clairement.</q>

Positionner des blocs dans une page

  • Lorsque l'on ne précise aucun positionnement, les blocs se placent les uns au-dessous des autres
  • Il est possible de positionner un bloc en utilisant :
    • Un bloc flottant : propriété float déjà rencontrée
    • La propriété position

Positionner avec la propriété position

  • La propriété position peut prendre les valeurs : static (défaut), absolute, fixed ou relative
  • absolute nécessite de définir la distance aux bords de la page avec les propriétés : left, right, top et bottom
  • fixed se définit comme absolute mais l'élément reste fixé à l'écran
  • relative permet un déplacement du bloc depuis sa position initiale dans son père

La propriété display

  • La propriété display modifie l'affichage standard d'un élément
  • display peut prendre les valeurs : none, block ou inline-block
  • none supprime l'affichage d'un élément
  • block transforme un élément en bloc (ex : la balise <a>)
  • inline-block permet aux blocs de se placer les uns à côté des autres (ex : balise <nav> à gauche ou à droite)
